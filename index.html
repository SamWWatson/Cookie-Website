Executive Summary

Cookies are small data files stored on a user's device to improve website functionality, user experience, and security. They help in remembering preferences, session management, and analytics tracking.

Key Terms

  • Session Cookies: Temporary cookies erased when the browser is closed.
  • Persistent Cookies: Remain on the device for a set duration.
  • First-Party Cookies: Created by the website being visited.
  • Third-Party Cookies: Created by external services for tracking and advertising.
  • Secure Cookies: Transmitted only via HTTPS.
  • HTTP-Only Cookies: Not accessible via JavaScript.
